About project

The idea of this project was born reading some articles about New Year's resolutions for a Web Developer. In my job I don't see new things very often 'cause of the pressure, go-live date, laziness and it's strange because the web is so fizzy and it's easy to be obsolete.

So my New Year's resolutions for 2012 are:

  • Learn Dojo Toolkit
  • Use jQuery in a better way (stop with 10000 lines domready, and spaghetti pattern)
  • Forget bad-practise
  • Get better at what I know
  • Share with others the good things I know
  • Become a Front-End Developer Rockstar
  • Be calm
  • Be calm
  • Be calm
